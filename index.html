Grid(un)Lock

A frustration free grid system.

Getting Started:

To get started, download the css file and link it to your webpage.

CSS 809b CSS (Minified) 634b

Usage

To use the grid, structure your content like this: 

        
          <section class="grid-container">
            <div class="row">
              <div class="col col-12"></div>
            </div>

To have multiple columns, just add multiple column divs. These must add up to 12. 

        
          <section class="grid-container">
            <div class="row">
              <div class="col col-6"></div>
              <div class="col col-6"></div>
            </div>
